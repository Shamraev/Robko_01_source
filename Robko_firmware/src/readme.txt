Прошивка для робота "Робко 01" с 5 степенями свободы
Написано в Visual Studio Code v.1.4 с дополнениями C++, PlatformIO 1.10 (нужно установить Python 3.8+ с включенной галочкой включить в глобальные переменные python);
Шамраев А.Д. 2020
************************************************************
Вся работа находится в папке '/src'

Пины управления находятся в '/src/src/RobkoPinOut.h'
Иные параметры для робота находятся в '/src/src/RobkoTypes.h'

*************Прошивка с помощью Arduino IDE****************
Для прошивки с помощью Arduino IDE содерживое папки '/scr': ('src', 'readme.txt', 'Robko_firmware.ino')
переместить в новую папку 'Robko_firmware'. В этой папке запустить файл 'Robko_firmware.ino' с помощью Arduino IDE. Прошить МК.
Загрузка в МК проверена на Arduino IDE 1.8.5.
