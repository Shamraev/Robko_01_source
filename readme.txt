Исходники для разработки программной части системы упрпавления
для робота "Робко 01" с 5 степенями свободы
Шамраев А.Д. 2019/2020
[плейлист видео](https://youtube.com/playlist?list=PLJ-_6HNRnl0PleVZsJ_78o03qlgVmuRIi)
*********************************************************************************
В каждой нижеперечисленной папке находятся пояснения в файле 'readme.txt'

В 'Programm' находятся исходники для десктопного приложения (для ПК)
среда разработки - Visual Studio C# 2017

В 'Robko_firmware' находятся исходники для прошивки микроконтроллера (для МК)
среда разработки - Visual Studio Code v.1.4 с дополнениями Arduino, C++; 
загрузка в МК с помощью Arduino IDE 1.8.5

В 'robko_PCB_arduino_shield' находятся исходники для проектирования платы управления
среда разработки - KiCad v.5.1.0

В 'Robko_workspase' находятся исходники для расчета операционного пространства
среда разработки - Matlab R2016a
